# bank_term_deposit_prediction
 Project Title:

“Bank Term Deposit Prediction using Machine Learning”

 Objective:

Predict customer subscription behavior

 Models Used:
Logistic Regression (baseline)

Random Forest (improved model)

Evaluation:

Accuracy: 80%+

Confusion Matrix analysis

Recall improvement focus

Key Insights:

Call duration is most important factor

Some potential customers are missed

Model is useful but can be optimized

Business Impact:

Helps bank target right customers

Improves marketing efficiency

Reduces wasted calls
